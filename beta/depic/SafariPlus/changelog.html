Safari Plus Changelog

1.7~b4 (May 1, 2019)

  • New Features (see depiction for more details):
    • Suggestion Insert Button
    • Tab Manager (Not fully done yet)

    Improvements:
    • Support for A12 / arm64e
    • Merged fixes from 1.6.8
    • Show an when a download is attempted to be started under cellular data while the 'Only Download on Wifi' option is enabled
    • Renamed 'Second 'Open in new Tab' option to 'Both Tab Open Actions' and changed it's behaviour accordingly
    • FullSafari compatibility has been restoredy
    • Updated arabic localization, thanks to iA7myd

1.7~b3 (April 18, 2019)

  • Improvements:
    • Several fixes for RTL languages (related to locked tabs and toolbar reordering)
    • Fix stupid oversight that caused lock button icons to not appear on 32 bit devices
    • Minor improvements to how locked tabs are handled

1.7~b2 (April 15, 2019)

  • Improvements:
    • Fixed an issue where the downloads button would not appear when the reordering options are disabled

1.7~b1 (April 15, 2019)

  • New Features (see depiction for more details):
    • Locked Tabs
    • Biometric Protection
    • Custom Start Site
    • Show Tab Count
    • Toolbar Reordering Options
    • Top Bar Reading Button Color
    • Top Bar Tab Bar Close Button Color
    • Tab Switcher Toolbar Background Color

    Improvements:
    • Reworked settings, added killswitch
    • Fixed status bar notifications being too big on some devices
    • Fixed choppy animation when using the "Open in opposite mode" option
    • Fixed desktop mode being always on when "Desktop Mode Switch" is disabled inside settings (¯\_(ツ)_/¯)
    • Improved desktop mode switch to feel and look more like the other buttons
    • Switched to libCSColorPicker
    • Reworked Force HTTPS Exceptions and Pinned Locations preferences (seperate plists no longer needed)
    • "safaripluscolorprefs.plist" and "safariplusprefsOther.plist" are automatically merged into "safariplusprefs.plist" on first launch of Safari or Preferences
    • Reworked some color settings to be less hacky
    • Many more bug fixes and improvements (more than I can remember)

1.6.7 (February 25, 2019)

  • Enhanced Downloads:
    • Fix an issue where pressing the video download button would sometimes crash on iOS 12

    Action Addons:
    • Browsing mode switch actions should now work properly again
